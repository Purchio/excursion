# Renaming the repo to `piano-coach`

The live app is at **https://piano-coach.vercel.app/** — that URL does not change when you rename the GitHub repo.

## Rename on GitHub

1. Open [github.com/Purchio/excursion/settings](https://github.com/Purchio/excursion/settings)
2. Change **Repository name** from `excursion` → `piano-coach`
3. Click **Rename**

## After rename

| Thing | URL |
|---|---|
| Live app (unchanged) | https://piano-coach.vercel.app/ |
| GitHub repo | github.com/Purchio/piano-coach |

Update your local clone if needed:

```bash
git remote set-url origin https://github.com/Purchio/piano-coach.git
```
